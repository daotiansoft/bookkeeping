**家庭记账系统 - 完整开发文档**

**一、项目概述**

一个基于 PyQt5 开发的本地桌面记账软件，提供完整的收入支出管理、分类管理、数据统计和报表打印功能。所有数据本地存储，无需联网，保护隐私。

**二、技术栈**

**2.1 核心技术**

| 技术  | 版本  | 用途  |
| --- | --- | --- |
| **Python** | 3.7+ | 主要开发语言 |
| **PyQt5** | 5.15+ | GUI图形界面框架 |
| **SQLite3** | 3.x | 本地数据库 |
| **Matplotlib** | 3.5+ | 图表绘制（柱状图、饼状图） |

**2.2 依赖安装**

bash

pip install PyQt5 matplotlib

**三、数据库设计**

**3.1 表结构**

**categories（分类表）**

| 字段  | 类型  | 说明  |
| --- | --- | --- |
| id  | INTEGER | 主键，自增 |
| name | TEXT | 分类名称，唯一 |
| type | TEXT | 类型：income/expense |
| created_at | TIMESTAMP | 创建时间 |

**transactions（交易记录表）**

| 字段  | 类型  | 说明  |
| --- | --- | --- |
| id  | INTEGER | 主键，自增 |
| category_id | INTEGER | 外键，关联分类表 |
| amount | REAL | 金额  |
| type | TEXT | 类型：income/expense |
| description | TEXT | 描述  |
| date | DATE | 交易日期 |
| created_at | TIMESTAMP | 创建时间 |

**3.2 默认分类**

**收入类**：工资、奖金、投资  
**支出类**：餐饮、交通、购物、娱乐、医疗、教育

**四、功能模块详解**

**4.1 记账录入模块**

**界面布局**：左右分屏（1:1比例）

- **左侧**：记账表单
- **右侧**：本月记录列表 + 本月统计

**功能清单**：

- 选择交易类型（支出/收入）
- 动态加载对应分类
- 金额输入（自动验证，只允许数字）
- 可选备注信息
- 日期选择（日历控件）
- 保存记录（自动刷新右侧数据）

**本月统计**：

- 显示本月总收入（绿色）
- 显示本月总支出（红色）
- 显示本月结余（正数绿色/负数红色）
- 实时更新

**本月记录列表**：

- 显示本月所有交易记录
- 列：编号、日期、类型、分类、金额
- 保存后自动刷新

**4.2 记录查询模块**

**查询条件**（同一行布局）：

- 开始日期
- 结束日期
- 交易类型筛选（全部/收入/支出）
- 分类筛选（动态加载）

**统计信息栏**：

- 总收入（绿色）
- 总支出（红色）
- 结余（正数绿色/负数红色）
- 记录数
- 批量删除按钮（红色）

**数据表格**：

- 列：编号、日期、类型、分类、金额(¥)、备注
- 支持多选（Ctrl/Shift）
- 交替行颜色
- 自适应列宽

**操作功能**：

- **查询**：根据条件筛选记录
- **重置**：恢复默认查询条件
- **打印**：生成报表并打印
- **批量删除**：删除选中的多条记录

**4.3 统计分析模块**

**统计条件**（同一行布局）：

- 自定义时间区间选择器
- 快捷筛选：今日、本周、本月、本年
- 生成统计按钮

**左右分屏布局**（各占50%）：

**左侧 - 统计摘要**

显示完整的财务报告：

- **统计区间信息**：起止日期、统计天数
- **收入情况**：
    - 总收入
    - 收入明细（各分类金额和占比）
- **支出情况**：
    - 总支出
    - 支出明细（各分类金额和占比）
- **结余情况**：
    - 净结余
    - 支出占收入比例
    - 健康度评估（健康/超支）

**右侧 - 统计图表**

- **柱状图**：各分类支出金额对比
    - 无金钱符号，仅显示数字
    - 渐变色柱状图
    - 数值标签
- **饼状图**：支出构成比例
    - 百分比显示
    - 彩色分区
    - 图例说明

**4.4 分类管理模块**

**添加分类**：

- 分类名称输入
- 分类类型选择（支出/收入）
- 添加按钮

**分类列表**：

- 列：编号、分类名称、类型
- 支持多选（Ctrl/Shift）
- 显示所有分类

**操作功能**：

- **批量删除**：
    - 删除选中的多个分类
    - 自动检测是否有交易记录
    - 有记录的分类无法删除并提示
    - 显示详细删除结果

**4.5 打印功能**

**打印内容**：

- 报表标题和生成时间
- 查询期间
- 统计摘要（总收入、总支出、结余、记录数、支出占比）
- 交易记录表格

**样式特点**：

- 专业报表格式
- 收入金额绿色显示
- 支出金额红色显示
- 包含页脚说明

**八、使用流程**

**8.1 初始化**

1.  运行 python main_window.py
2.  自动创建数据库和默认分类
3.  进入记账录入界面

**8.2 日常记账**

1.  选择交易类型 → 选择分类 → 输入金额 → 填写描述 → 确认日期
2.  点击"保存记录"
3.  右侧自动显示本月记录和统计

**8.3 查询记录**

1.  切换到"记录查询"标签页
2.  设置查询条件（日期、类型、分类）
3.  点击"查询"查看结果
4.  可打印或批量删除

**8.4 统计分析**

1.  切换到"统计分析"标签页
2.  选择统计区间或使用快捷按钮
3.  点击"生成统计"
4.  查看统计摘要和图表

**8.5 分类管理**

1.  切换到"分类管理"标签页
2.  添加新分类或批量删除现有分类
3.  系统自动更新所有相关界面
